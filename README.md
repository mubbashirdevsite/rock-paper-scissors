# ✊ Rock Paper Scissors Game

A modern interactive **Rock Paper Scissors game** built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- ✊ Rock, 📄 Paper, ✂️ Scissors gameplay
- 🤖 Computer random choice
- 🧠 Game modes:
  - Best of 3
  - Best of 5
  - Endless Mode
- 📊 Score tracking (User, Computer, Draw)
- 🔢 Round counter
- 🏆 Final winner announcement
- 🔄 Restart / Play Again option
- 📱 Fully responsive UI
- ✨ Smooth animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
